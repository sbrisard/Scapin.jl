Hooke{d,T}

Isotropic, linear elastic material.

  • d ‚Äî number of spatial dimensions
  • T ‚Äî scalar type
  • Œº::T ‚Äî shear modulus
  • ŒΩ::T ‚Äî Poisson ratio

Regardless of the number of space dimensions, d, the stress-strain relationship reads

œÉ = Œª‚ãÖtr(Œµ)I + 2Œº‚ãÖŒµ.
Material stability

Material stability requires that Œº > 0 and -1 < ŒΩ < 1/2; these conditions are not enforced here. In other words, unstable materials can be defined.

In the current implementation, d = 2 refers to plane strain elasticity. For plane stresses, the true Poisson ratio ŒΩ should be replaced with the fictitious ratio ŒΩÃÉ = ŒΩ / (1 + ŒΩ).
