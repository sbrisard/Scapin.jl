Scapin.Hooke — Type

Isotropic, linear elastic material.

Hooke{T, DIM}(μ::T, ν::T)

Create a new instance with shear modulus μ and Poisson ratio ν.

Material stability

Material stability requires that μ > 0 and -1 < ν < 1/2; these conditions are not enforced here. In other words, unstable materials can be defined.

Plane stresses vs. plane strains

In the current implementation, DIM = 2 refers to plane strain elasticity. For plane stresses, the true Poisson ratio ν should be replaced with the fictitious ratio ν̃ = ν / (1 + ν).

Scapin.bulk_modulus — Method
bulk_modulus(C::Hooke)

Return the bulk modulus κ for the specified Hooke material.

For plane strain elasticity

κ = μ / (1 - 2ν),

and, for 3D elasticity

κ = 2/3 μ (1 + ν) / (1 - 2ν).
